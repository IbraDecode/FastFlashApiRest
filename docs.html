Fast Flash API

Dokumentasi lengkap untuk menggunakan Fast Flash API - REST API yang powerful dan mudah digunakan

Getting Started

Fast Flash API adalah REST API yang menyediakan berbagai layanan mulai dari AI, download media, pencarian, tools, dan masih banyak lagi. API ini dirancang untuk mudah digunakan dan terintegrasi dengan aplikasi Anda.

Base URL

https://your-domain.com

Format Response

Semua response API menggunakan format JSON dengan struktur berikut:

{
  "status": true,
  "creator": "By Ibra Decode",
  "result": {
    // Data response
  }
}

HTTP Methods

API ini menggunakan method HTTP GET untuk semua endpoint. Parameter dikirim melalui query string.

Rate Limiting

Untuk menjaga performa server, API ini menerapkan rate limiting. Pastikan Anda memiliki API key yang valid untuk akses unlimited.

Authentication

Fast Flash API menggunakan API key untuk autentikasi. API key harus disertakan dalam setiap request sebagai parameter query.

Cara Menggunakan API Key

Tambahkan parameter apikey pada setiap request:

GET /api/endpoint?apikey=YOUR_API_KEY¶m1=value1

Usage Examples

JavaScript (Fetch API)

// Download TikTok Video
const apiKey = 'YOUR_API_KEY';
const url = 'https://tiktok.com/video-url';

fetch(`https://your-domain.com/download/tiktok?apikey=${apiKey}&url=${encodeURIComponent(url)}`)
  .then(response => response.json())
  .then(data => {
    if (data.status) {
      console.log('Video data:', data.result);
    } else {
      console.error('Error:', data.message);
    }
  })
  .catch(error => console.error('Request failed:', error));

Python (Requests)

import requests

# Search YouTube
api_key = 'YOUR_API_KEY'
query = 'tutorial programming'

response = requests.get(f'https://your-domain.com/search/youtube?apikey={api_key}&q={query}')
data = response.json()

if data['status']:
    print('Search results:', data['result'])
else:
    print('Error:', data.get('message', 'Unknown error'))

PHP (cURL)

<?php
// AI Chat with Gemini
$apiKey = 'YOUR_API_KEY';
$text = 'Halo, apa kabar?';

$url = "https://your-domain.com/ai/gemini?apikey=" . urlencode($apiKey) . "&text=" . urlencode($text);

$response = file_get_contents($url);
$data = json_decode($response, true);

if ($data['status']) {
    echo "AI Response: " . $data['result']['response'];
} else {
    echo "Error: " . $data['message'];
}
?>

Node.js (Axios)

const axios = require('axios');

// Instagram Stalker
const apiKey = 'YOUR_API_KEY';
const username = 'instagram_username';

axios.get(`https://your-domain.com/stalk/instagram?apikey=${apiKey}&username=${username}`)
  .then(response => {
    const data = response.data;
    if (data.status) {
      console.log('Profile data:', data.result);
    } else {
      console.error('Error:', data.message);
    }
  })
  .catch(error => console.error('Request failed:', error));

Error Handling

HTTP Status Codes

Status Code Description
200 Success - Request berhasil diproses
400 Bad Request - Parameter tidak valid atau kurang
401 Unauthorized - API key tidak valid atau tidak ada
429 Too Many Requests - Rate limit terlampaui
500 Internal Server Error - Error pada server

Error Response Format

Ketika terjadi error, API akan mengembalikan response dengan format berikut:

{ "status": false, "message": "Error description", "error": "Detailed error information (optional)" }

Common Errors

Missing API Key

Terjadi ketika parameter apikey tidak disertakan dalam request

{ "status": false, "message": "API key is required" }

Invalid URL Parameter

Terjadi ketika URL yang diberikan tidak valid atau tidak didukung

{ "status": false, "message": "Invalid URL format or unsupported platform" }

Rate Limit Exceeded

Terjadi ketika jumlah request melebihi batas yang diizinkan

{ "status": false, "message": "Rate limit exceeded. Please try again later." }

Best Practices

  • Selalu periksa status response sebelum memproses data
  • Implementasikan retry logic untuk error sementara (5xx)
  • Gunakan timeout yang wajar untuk request
  • Log error untuk debugging dan monitoring
  • Validasi parameter sebelum mengirim request
