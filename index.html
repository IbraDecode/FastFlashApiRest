Getting Started

Selamat datang di dokumentasi Fast Flash API. API ini menyediakan berbagai layanan mulai dari AI, downloader, search, tools, dan masih banyak lagi.

Base URL

https://api.xmsbra.web.id/api

Authentication

Semua request memerlukan API key yang valid. Tambahkan parameter apikey pada setiap request:

GET /api/ai/openai?apikey=YOUR_API_KEY&text=hello

Response Format

Semua response menggunakan format JSON dengan struktur berikut:

{
  "status": true,
  "creator": "By Ibra Decode",
  "result": {
    // Response data here
  }
}
