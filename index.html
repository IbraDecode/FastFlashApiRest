Memulai

Selamat datang di dokumentasi Fast Flash API. API ini menyediakan berbagai layanan mulai dari AI, downloader, pencarian, tools, dan masih banyak lagi.

Base URL

https://your-api-domain.com/api

Autentikasi

Semua permintaan memerlukan API key yang valid. Tambahkan parameter apikey pada setiap permintaan:

GET /api/ai/openai?apikey=YOUR_API_KEY&text=halo

Format Respons

Semua respons menggunakan format JSON dengan struktur berikut:

{
  "status": true,
  "creator": "By Ibra Decode",
  "result": {
    // Data respons di sini
  }
}
